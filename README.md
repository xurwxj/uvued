# Bootup Guide

## Project setup
```
yarn install
```

## init config file
```
cp demo.js conf.js
```

### Compiles and hot-reloads for development
```
yarn run ssr:serve
```

### Compiles and minifies for production
```
yarn run ssr:build
```

### Runs for production
```
yarn run ssr:start
```
